# postcss

## 插件说明

\*. 文件名 .postcssrc.js

\*.插件

```js
/**
 * postcss配置规则写在package.json文件内部
 * 插件说明：
 * 1. autoprefixer    === 添加浏览器私有前缀   vue-cli 内置
 * 2. postcss-import  === @import "bar.css" 导入语法
 * 3. postcss-nested  === scss嵌套规则
 * 3. postcss-assets  === 图像处理（Base64 编码等）
 * 4. cssnext         === var 变量等下一代css样式写法
 * 4. postcss-mixins  === mixins
 * 4. postcss-simple-vars  === $变量
 * 4. postcss-comment  === 内嵌注释
 * 4. postcss-precss  === scss 部分语法
 * * rucksack-css     === 提供函数
 * * Stylelint        === 规则校验
 * * css-mqpacker     === 处理媒体查询
 * * cssnano          === 压缩
 * *. postcss-precss  === scss 部分语法 本项目使用插件
 */
```
